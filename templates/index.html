About

Ever had that problem when you know of the concept yet can't recall the word? Thesaurus are usually pretty poor at this task because it's built to link individual words to each other. Not a rough description to a single word.

This is a solution built around WordNet through NLTK and TextBlob. It utilizes Tf-idf implemented by Elasticsearch to index each term by their definitions.
